# AI Notes Digitizer

A Streamlit app that digitizes handwritten notes using GPT OCR.

## Features
- Upload images or PDF pages
- Preprocess images (grayscale, deskew, denoise, binarize)
- Rotate image manually
- Extract text with GPT OCR
- Edit, copy, or download digitized text

## How to use
1. Clone or download the repository
2. Install dependencies from `requirements.txt`
3. Add your OpenAI API key in `.env`
4. Run `streamlit run app.py`
